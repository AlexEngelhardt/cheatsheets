### Movement

`gg`
: Goto line 0

`10G`
: Goto line 10

`GG`
: Goto last line

cmd | description
--- | -----------
`gg` | Goto line 0
`10G` | Goto line 10
`GG` | Goto last line

### Undo

`u`
: Undo last

`U`
: Undo everything you did on this line

`Ctrl-R`
: Redo
